|3.3.1|Remoción de silencios|Implementar técnicas de detección y eliminación de silencios al inicio, final o interior de las grabaciones para conservar únicamente regiones relevantes de voz. Mejora la calidad del análisis posterior.|COD|
|3.3.2|Normalización de amplitud|Ajustar el nivel de amplitud de las grabaciones para reducir variaciones debidas a intensidad de grabación o volumen del hablante. Esto ayuda a que el modelo aprenda características patológicas y no diferencias de adquisición.|COD|
|3.3.3|Segmentación en ventanas de análisis|Dividir cada señal de audio en segmentos temporales de longitud fija para permitir análisis locales y aumentar la cantidad de muestras utilizables durante el entrenamiento.|COD|
|3.3.4|Resampling y unificación de frecuencia de muestreo|Convertir todas las grabaciones a una frecuencia de muestreo uniforme para garantizar compatibilidad entre etapas de procesamiento y comparabilidad entre muestras.|COD|
|3.4.1|Implementación STFT / espectrograma|Implementar la Short-Time Fourier Transform para obtener espectrogramas que representen la evolución temporal del contenido frecuencial de la voz. Servirá como baseline clásico de análisis tiempo-frecuencia.|COD|
|3.4.2|Implementación CWT con Morlet|Aplicar Continuous Wavelet Transform utilizando la wavelet Morlet para analizar componentes oscilatorias y variaciones locales de frecuencia en la señal de voz. Su excelente resolución tiempo-frecuencia la hace adecuada para señales biomédicas no estacionarias.|COD|
|3.4.3|Implementación DWT con Daubechies-4|Implementar Transformada Wavelet Discreta utilizando la wavelet Daubechies-4 para realizar descomposición multirresolución eficiente de la señal de voz y extraer información en diferentes escalas frecuenciales.|COD|
|3.4.4|Implementación DWT con Symlet|Implementar Transformada Wavelet Discreta utilizando wavelets Symlet para preservar mejor la morfología temporal y reducir distorsiones de fase durante el análisis de señales vocales.|COD|
|3.4.5|Implementación EMD-HHT (modos empíricos + Hilbert)|Implementar descomposición empírica en modos intrínsecos seguida del análisis de Hilbert-Huang para estudiar modulaciones instantáneas y componentes adaptativas de la señal de voz.|COD|
|3.4.6|Ajuste de parámetros de representaciones|Evaluar parámetros relevantes de las representaciones tiempo-frecuencia, como tamaño de ventana, overlap, escalas wavelet y niveles de descomposición, para optimizar calidad descriptiva y desempeño de clasificación.|COD|
|3.4.7|Visualizaciones comparativas sano vs PD|Generar representaciones visuales comparativas entre sujetos sanos y sujetos con Parkinson utilizando las distintas transformadas implementadas. Estas figuras permiten identificar patrones diferenciales y comunicar resultados.|COD|
|3.5.1|Estadísticos sobre cada representación|Extraer descriptores numéricos a partir de las representaciones tiempo-frecuencia, como energía, media, desviación estándar, entropía o métricas espectrales relevantes. Estos valores constituyen los features de entrada al clasificador.|COD|
|3.5.2|Construcción y guardado de matriz de features|Organizar todos los descriptores extraídos en una matriz estructurada junto con sus etiquetas correspondientes. Guardar el resultado en formatos reutilizables para evitar reprocesamiento innecesario.|COD|
|3.5.3|Normalización y estandarización de features|Aplicar técnicas de escalado y normalización sobre los vectores de características para mejorar estabilidad numérica y desempeño de los clasificadores, especialmente en modelos sensibles a escala como SVM.|COD|
|3.6.1|Implementación SVM-RBF|Entrenar y evaluar un clasificador Support Vector Machine con kernel radial basis function para separar sujetos sanos y pacientes con Parkinson a partir de los features extraídos.|COD|
|3.6.2|Implementación Random Forest|Implementar un clasificador Random Forest basado en múltiples árboles de decisión para comparar desempeño, robustez e interpretabilidad respecto a SVM.|COD|
